
-- SUMMARY --

The Issuu API module contains a full implementation of all the issuu.com APIs.
This module will not add any additional functionality (e.g.: Fields, ...),
it sole purpose is to provide an easy way of interacting with the issuu.com API.

This module is part of the Issuu integration project:
  http://drupal.org/project/issuu
To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/issuu

-- REQUIREMENTS --

The Issuu API module depends on the http_client module and requires the php cURL
extension to be installed.

To find out more on how to install and configure the php cURL extension see:
http://www.php.net/manual/en/curl.setup.php
